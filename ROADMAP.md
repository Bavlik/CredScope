# Roadmap

The following items are planned and are not implemented features:

- Further credential-classification refinement.
- Additional secret-scanner adapters behind the scanner-neutral interface.
- Kubernetes configuration analysis.
- Additional CI provider support.
- Signed release binaries and stronger artifact provenance.
- Additional package-manager distribution after the planned WinGet submission.

Roadmap items do not imply a delivery date. WinGet packaging is prepared but remains unavailable until a release is published and Microsoft accepts the manifest. CredScope currently supports Gitleaks import, Docker Compose, and GitHub Actions static analysis only.
