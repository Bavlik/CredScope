# Support

Use GitHub Discussions if enabled for usage questions and design discussion. Use the issue templates for reproducible bugs and scoped feature proposals. The project is community-supported and does not offer a service-level agreement.

Do not post credentials, unredacted scanner reports, private repository content, or vulnerability details in public support channels. Follow [SECURITY.md](SECURITY.md) for security reports.

## Before Opening an Issue

Before creating a new issue, please:

- Search existing issues and discussions.
- Make sure you are using the latest release.
- Include clear reproduction steps and relevant environment details.
- Redact any sensitive information before sharing logs or reports.
